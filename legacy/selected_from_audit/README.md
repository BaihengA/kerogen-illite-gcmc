# Kerogen–Illite 8 nm Pore + RASPA2 GCMC

Version: **v33 / 0.33.0**

This repository-ready package continues the existing kerogen roughness workflow and builds an **8 nm mirrored composite slit pore** from the final kerogen–illite wall generated by V31.

## Scientific geometry

For each case:

- `RMS_0p000`
- `RMS_0p300`
- `RMS_0p600`
- `RMS_0p900`

The lower wall is read from:

```text
ILLITE_BELOW_RAW_FINAL_KEROGEN_V31_THICKER_SAFE/{case}/
{case}_EXACT_kerogen_plus_illite.gro
```

The exact RAW kerogen reference is read from:

```text
KEROGEN_ONLY_FINALS/RAW_KEEP_COORDINATES/
{case}_kerogen_only_raw_from_final.gro
```

The upper wall is generated by an exact z-mirror of the **whole kerogen + illite composite wall**. The default 8 nm pore width is defined between the two mean rough-surface planes:

```text
upper illite
====================
upper kerogen
\/\/\/\/\/\/\/\/\/   rough surface faces downward

        8.0 nm
  mean-plane definition

/\/\/\/\/\/\/\/\/\   rough surface faces upward
lower kerogen
====================
lower illite
```

The build report also records local grid-gap statistics (`min`, `p05`, `mean`, `p95`, `max`) so that roughness-dependent local aperture is not hidden by the nominal 8 nm definition.

## Run order on Windows

```text
48_BUILD_8NM_COMPOSITE_PORES.bat
49_PREPARE_RASPA2_GCMC_8NM.bat
50_RUN_RASPA2_GCMC_8NM.bat
51_COLLECT_GCMC_COUNTS_8NM.bat
```

Or run the complete chain:

```text
52_ALL_8NM_BUILD_PREP_RUN_COLLECT.bat
```

## Main configuration

Edit:

```text
8nm_pore_raspa2_config.json
```

Key fields:

```json
"pore_width_nm": 8.0,
"pore_width_definition": "mean_surface_plane",
"temperature_K": 353.15,
"pressure_MPa": [20.0]
```

`353.15 K / 20 MPa` is only the current default template. Replace these values with the actual formation temperature and pressure of the target reservoir before reporting final GCMC molecule numbers.

## GCMC outputs

Prepared RASPA2 cases are written to:

```text
RASPA2_GCMC_8NM_V33/
```

Collected molecule counts are written to:

```text
RASPA2_GCMC_8NM_V33/GCMC_CH4_H2O_MOLECULE_COUNTS.csv
```
